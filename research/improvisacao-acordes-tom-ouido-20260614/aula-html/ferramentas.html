Ferramentas interativas

Hub central para visualizar campos harmônicos, transpor progressões, prever acordes prováveis, treinar funções e converter cifras para números — use durante o estudo e no palco (tablet).

Como usar este hub

Cada ferramenta abaixo corresponde a um conceito do curso. Use-as para confirmar o que o ouvido suspeita, não para substituir o treino auditivo.

  • Durante estudo: teste hipóteses antes de abrir a cifra
  • Na preparação: monte charts numéricos de repertório
  • No palco (com moderação): transposição rápida se cantor mudar tom

1. Campo harmônico interativo

Visualize os 7 graus diatônicos de qualquer tom: numeração romana, NNS, função (T/SD/D) e sensação auditiva. Base do módulo 03.

Campo harmônico — escolha a tônica

2. Transpositor de progressões

Selecione uma progressão-modelo (Axis, Doo-wop, ii–V–I), escolha tom original e novo tom. Veja acordes convertidos + números + graus romanos. Essencial do módulo 07.

Transpor progressão

3. Preditor de acordes (Markov)

Com base em estatísticas de progressões pop, bossa e samba: dado o grau atual, quais são os próximos mais prováveis? Use como aposta inicial — confirme pelo ouvido. Módulo 06.

Próximo acorde provável

4. Quiz de funções harmônicas

Teste rápido: classifique acordes como Tônica, Subdominante ou Dominante. Treino de reflexo para o módulo 04.

Quiz — funções T · SD · D

Acertos: 0

5. Conversor cifra → números (NNS)

Cole acordes separados por espaço ou hífen. O sistema converte para Nashville Number System assumindo o tom selecionado como tônica. Primeiro passo do workflow de transposição (módulo 07).

Cifra → números

Workflow completo: (1) converter cifra → números aqui, (2) declarar novo tom, (3) usar transpositor ou mentalmente mapear números → novas letras.
