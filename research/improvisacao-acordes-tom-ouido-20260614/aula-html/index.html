Por que não é magia

Músicos experientes executam um pipeline mental treinado por anos. Cada etapa é uma competência isolada que se estuda, pratica e automatiza:

  1. Entrada auditiva — voz, baixo, harmonia implícita
  2. Tônica — centro tonal da frase
  3. Baixo = raiz — na pop/MPB, ~80% das vezes
  4. Função — repouso (T), movimento (SD), tensão (D)
  5. Qualidade — maior, menor, 7, °, maj7…
  6. Antecipação — grau provável pelo contexto + estatística
  7. Transposição — pensar em números, não letras
  8. Voicing — forma enxuta que deixa espaço ao cantor
O segredo central: pensar em função (I, V, vi, IV), não em letras fixas (C, G, Am, F). O Nashville Number System nasceu para session musicians chartarem músicas na primeira audição.

Como usar este curso

  • Siga a ordem das fases — cada módulo assume o anterior.
  • Faça todos os exercícios no instrumento, cantando os graus em voz alta.
  • Use Ferramentas para transpor e visualizar campos harmônicos.
  • Marque cada módulo como concluído — o progresso fica salvo no navegador.
  • Reserve 30–45 min/dia; consistência vence maratonas de fim de semana.
