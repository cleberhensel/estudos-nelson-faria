Integrar em outras páginas

Inclua o CSS + JS e monte os componentes em qualquer container. Eventos customizados sobem pelo DOM (practice:beat, practice:complete, etc.). 

<link rel="stylesheet" href="tools/practice-tools.css">
<div id="metro-root"></div>
<script src="tools/practice-tools.js"></script>
<script>
  const metro = new PracticeTools.Metronome({ container: '#metro-root', bpm: 90 });
  metro.on('beat', ({ beat, accent }) => { /* sincronizar tab, validador, etc. */ });

  const timer = new PracticeTools.Timer({ container: '#timer-root', durationSec: 180 });
  timer.on('complete', () => metro.stop());
</script>
