# PLN-03 — Currículo de Aulas Extensas

> Catálogo de aulas no formato **teoria musical brasileira** (1500–4500 palavras cada)  
> Cada entrada = unidade publicável com campo harmônico, escalas, história, análise e exercícios

---

## Convenções

| Campo | Significado |
|-------|-------------|
| **ID** | Código único (ex.: NF-A1) |
| **Densidade** | Palavras-alvo · exemplos musicais · trechos analisados |
| **Pré-req.** | Aulas que devem preceder (flexível na trilha PLN-04) |
| **Entregável** | Artefato concreto ao fim da aula |

## Template de redação (checklist por aula)

Cada aula publicada deve conter:

- [ ] Por que estudar isto (link Nelson Faria)
- [ ] Contexto histórico
- [ ] Campo harmônico (mapa + progressões)
- [ ] Escalas e recursos melódicos
- [ ] ≥3 análises de trechos com cifra/função
- [ ] Técnicas do repertório (numeradas)
- [ ] ≥4 exercícios práticos
- [ ] Síntese + para ir além
- [ ] Timestamp(s) do vídeo fonte
