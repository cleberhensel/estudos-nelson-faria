# estudos-nelson-faria

Hub de didática, estudos e aprofundamento musical: currículo Nelson Faria, teoria, research e ferramentas de estudo.

## Estrutura

| Pasta | Conteúdo |
|---|---|
| `nelson-faria/` | PLN, módulos (braço, ginásio, piloto), transcrições |
| `theory/plan/` | Planos editoriais de teoria (Tocaí) |
| `theory/content/` | Aulas JSON, glossário, anotações, currículo |
| `research/` | Discovery, deep-research, relatórios |
| `repertoire/cifras/` | Cifras usadas como material de estudo |
| `tools/transcrever-violao/` | Ferramenta local de transcrição |

## Abrir módulos HTML

Na raiz do repo:

```bash
npx serve nelson-faria/modulos/escalas-arpejos-treino/pages
```

Ou abrir diretamente no browser (file://) a partir de `nelson-faria/modulos/`.
